# Support

## Getting Help

For usage questions, bug reports, and feature requests, use the GitHub issue tracker:

- https://github.com/Sophylax/envguard/issues

Before opening a new issue, check for an existing report or discussion that matches your problem.

## What to Include

Helpful support details:

- `envguard version`
- operating system and architecture
- Git version if the problem involves hooks or repo detection
- relevant `.envguard.yml` settings
- a minimal reproduction or sample command

## Support Expectations

`envguard` is maintained on a best-effort basis. Priority goes to:

- correctness bugs in local pre-commit scanning
- release and installation issues
- cross-platform regressions

Requests outside the project scope may be closed or redirected.
